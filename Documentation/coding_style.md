# Coding style

- Functions: lower_snake_case
- Variables: lower_snake_case
- Private variables: lower_snake_case_with_underscore_at_end_
- Member variables: lower_snake_case
- Constant variables: UPPER_CASE_SNAKE
- Classes: UpperCamelCase
- Pointers: after type not variable (Vector3* my_vector)
- Namespaces: lower_snake_case
- Files: lower_snake_case
